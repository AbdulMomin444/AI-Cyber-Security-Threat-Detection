# AI-Cyber-Security-Threat-Detection
Real-time AI-based Cyber Security Dashboard that detects anomalies, analyzes network traffic, and provides live threat analytics with visualization.

## ✨ Features
- Real-time packet monitoring (Scapy)
- AI anomaly detection model
- Live dashboard with analytics
- Threat classification (DDoS, Port Scan)
- IP geolocation tracking
- Email alert system
- Smart filtering & search (Phase 3)
- Export logs (CSV)

## 🛠 Tech Stack
- Python (Flask)
- Scapy
- Machine Learning (Isolation Forest)
- HTML, CSS, JavaScript
- Chart.js
